# AI Organization MVP Blueprint

## 0. Design Principles
- 日本語メイン + 英語キーワード併記
- 小さく始める（12〜16 agents）
- KPI-driven operation
- Human-in-the-loop for critical decisions

## 1. Org Layers
- **L0 Human CEO**: 最終承認 (Final Approval)
- **L1 PRIME AI (Chief AI Operator)**: 全体統括 (Orchestration)
- **L2 Division Leads**: 4事業部長
- **L3/L4 Managers & Agents**: 実行ユニット

## 2. MVP Staffing (14 agents)
### A. Content Division (4)
1. Content Lead (Manager)
2. Planning Agent (Trend +企画)
3. Production Agent (Script/Asset)
4. Publishing & Analytics Agent

### B. BizDev Division (4)
1. BizDev Lead (Manager)
2. Lead Generation Agent
3. Sales Outreach Agent
4. Customer Success Agent

### C. Tech Division (3)
1. Tech Lead (Manager)
2. Dev/Automation Agent
3. QA/Deploy Agent

### D. Operations Division (3)
1. Ops Lead (Manager)
2. Task & Calendar Agent
3. Comms & Data Ops Agent

## 3. Cadence
- Daily: Standup summary by PRIME AI
- Weekly: KPI review + bottleneck report
- Monthly: Org tuning (agent add/remove)

## 4. Decision Gates
- 1万円以上の支出: Human approval required
- 外部公開・契約・課金変更: always human approval
- 自動実行停止条件: 連続失敗3回 or KPI急落

## 5. 90-Day Milestones
- Day 1-14: 1事業部で試運転 (Content)
- Day 15-45: BizDev/Techを段階導入
- Day 46-90: Ops統合 + KPI最適化
