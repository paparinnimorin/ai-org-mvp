# Agent Staffing Model (MVP)

## Initial Allocation (14 agents)
- Content: 4
- BizDev: 4
- Tech: 3
- Ops: 3

## Required Positions
- Content Lead / Planning / Production / Publishing&Analytics
- BizDev Lead / LeadGen / Outreach / CS
- Tech Lead / DevAutomation / QADeploy
- Ops Lead / TaskCalendar / CommsData

## Substitutable Roles
- Planning <-> Publishing&Analytics（短期代替可）
- LeadGen <-> Outreach（初期は兼務可）
- QADeploy <-> DevAutomation（緊急時のみ）
- TaskCalendar <-> CommsData（負荷平準化時）

## Scale-up Triggers
- 2週連続でKPI 4/5達成 + backlog増加
- Content公開本数が目標超過し制作待ち増大
- Outreach返信増でSLA維持困難

## Scale-down / Merge Triggers
- 4週連続で業務量が下限未満
- 役割重複で品質影響なし
- 自動化率向上により手動介入が低下
