# Manager Cycle Midcheck Checklist

cycle中盤レビューの実施確認チェックリスト。

## 1) Progress
- [ ] cycle対象ticketの進捗を更新した
- [ ] blocked項目を分類して件数を記録した
- [ ] carry-over候補を洗い出した

## 2) Risk & Approval
- [ ] 重大リスク（impact高）を特定した
- [ ] 承認が必要な変更点を整理した
- [ ] decision logへの記録要否を判定した

## 3) Replan
- [ ] 次の48hアクションをowner付きで定義した
- [ ] 優先順位変更をtask boardに反映した
- [ ] 依存解消の期限を設定した

## 4) Publish
- [ ] midcheck reportを生成/保存した
- [ ] `docs/README.md` のリンク整合を確認した
- [ ] bundle checkを実行した
