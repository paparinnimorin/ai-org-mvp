# Manager Dispatch Checklist

## Pre-dispatch
- [ ] Goalが1文で明確
- [ ] Deliverablesがファイルパスで明記されている
- [ ] Todoが30分以内の粒度に分解されている
- [ ] Verifyコマンドが最低2つ（ticket-check, link-check）含まれる
- [ ] 依存関係（Blocked By）が明記されている

## Post-dispatch
- [ ] 生成物パスが期待場所にある
- [ ] Todoの`[x]`が実装と一致
- [ ] `docs/README.md` にリンクが追加されている
- [ ] checks実行結果がログ化されている
