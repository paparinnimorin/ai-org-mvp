# Governance Action Verification Checklist

- [ ] 対象 minutes を特定し、action ID一覧を確定した
- [ ] action register で各IDの現状態を確認した
- [ ] closure/reopen/escalation/scope-change/deferral の該当レポートを確認した
- [ ] evidence（成果物リンク・担当者・日付）を各IDで確認した
- [ ] KPI/リスク影響を確認し、必要な追跡項目を記録した
- [ ] 不整合の有無を判定し、Pass/Needs-Fixを決定した
- [ ] verification reportを保存し、参照リンクを共有した
