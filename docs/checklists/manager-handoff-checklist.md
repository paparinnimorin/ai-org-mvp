# Manager Handoff Checklist

## Before Handoff
- [ ] manager boardのTop 5タスクを最新化
- [ ] 当日追加のdecision/approvalログを確認
- [ ] blockerごとのownerと期限を明記

## Handoff Note Quality Gate
- [ ] Current Stateが具体的（ticket id/owner付き）
- [ ] Risks/Blockersにimpactとetaが含まれる
- [ ] Next 24h Actionsが優先順で3件以上
- [ ] Escalation Flagsが更新済み

## After Handoff
- [ ] `docs/reports/handoffs/` にhandoff note保存
- [ ] 必要時にmanager briefへ逆リンクを追記
