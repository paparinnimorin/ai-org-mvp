# Manager Cycle Kickoff Checklist

cycle開始前の最終確認用チェックリスト。

## 1) Scope & Goals
- [ ] cycle範囲（例: 047-050）を確定した
- [ ] 主要ゴールを3点以内で言語化した
- [ ] carry-over項目を抽出した

## 2) Ownership & Schedule
- [ ] 各ticketのownerを割り当てた
- [ ] 期限/マイルストーンを設定した
- [ ] 依存関係の順序を確認した

## 3) Artifacts Ready
- [ ] 必要テンプレートを準備した
- [ ] 必要CLI/チェック手順を確認した
- [ ] `docs/README.md` に参照リンクを整備した

## 4) Publish
- [ ] kickoff briefを生成/保存した
- [ ] task boardへ反映した
- [ ] kickoff時点のリスクを記録した
