# Governance Action Closure Checklist

## Metadata
- Minutes: `docs/reports/minutes/<minutes>.md`
- Action Register: `docs/reports/minutes/actions/<register>.md`
- Closure Report: `docs/reports/minutes/closures/<closure>.md`
- Reviewer: manager

## Pre-check
- [ ] minutesが確定済み
- [ ] action registerが最新化済み（owner/due/status）
- [ ] overdue/blocked項目が識別済み

## Validation
- [ ] Closed actionに証跡リンクがある
- [ ] Carry Over actionに次期限が設定されている
- [ ] Blocked actionに escalation owner と due がある

## Reporting
- [ ] closure reportを作成
- [ ] Summary（closed/open/blocked/carry over）を記載
- [ ] 次サイクルへの引継ぎ項目を明記

## Handoff
- [ ] manager brief / cycle checklistへ反映
- [ ] ticket status更新スクリプトを実行
- [ ] READMEリンクチェックを実行
