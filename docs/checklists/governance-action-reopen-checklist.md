# Governance Action Reopen Checklist

- [ ] Source minutes / action ID を特定した
- [ ] Reopen reason を分類した（Evidence Gap / Scope Change / Regression）
- [ ] 新ownerと新dueを設定した
- [ ] `docs/reports/minutes/reopens/` にreopen reportを作成した
- [ ] closure report への参照を追加した
- [ ] 次サイクルkickoffへの引継ぎ先を明記した
- [ ] `docs/ticket-status.md` 更新前提で関連ticketの完了状態を確認した
