# Weekly Close Checklist

金曜クローズ時の標準手順（Japanese main + English keywords）。

## 1) KPI Close
- [ ] `reports/weekly-kpi-snapshot.md` の数値を確定
- [ ] KPI入力漏れを `process/kpi-input-governance.md` で確認
- [ ] 前週比の変化点（+/-）を3点以内で要約

## 2) Governance Inputs
- [ ] Approval/Decision log の週内更新を確認
- [ ] 重大事項（High impact）を議題化
- [ ] 翌週の承認予定（approval queue）を整理

## 3) Risk & Safety
- [ ] `risk-register.md` のTop risksを更新
- [ ] Incident/near-miss の有無を確認
- [ ] Stop conditions該当の再点検

## 4) Action Lock
- [ ] 次週Top3 actionをowner/due付きで確定
- [ ] 未完了アクションのcarry-over理由を記録
- [ ] 週次レトロ実施日時を確定
