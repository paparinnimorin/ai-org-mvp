# Task Execution Checklist

## Before Start
- [ ] Goal/Deliverables/Todo を確認
- [ ] 依存ファイルの現状を確認
- [ ] 実装順序（smallest-first）を決定

## During Implementation
- [ ] 1 Todoごとに実装→確認を実施
- [ ] 仕様変更は process/template に反映
- [ ] path typo / link typo を都度確認

## Verification
- [ ] `python3 scripts/check_tickets.py` 実行
- [ ] 新規スクリプトを最低1回実行
- [ ] `python3 scripts/check_readme_links.py` 実行

## Completion
- [ ] docs/README.md へリンク反映
- [ ] Ticket Todo を [x] に更新
- [ ] Commit message を ticket番号付きで作成
