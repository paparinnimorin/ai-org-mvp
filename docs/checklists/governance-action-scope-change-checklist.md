# Governance Action Scope Change Checklist

## Pre-check
- [ ] source minutes を特定した
- [ ] 最新action registerを参照した
- [ ] close/reopen/escalationで解決不能であることを確認した

## Scope Design
- [ ] 変更理由（business / risk / dependency）を記録した
- [ ] deliverables変更（追加/削除）を記録した
- [ ] due変更（旧→新）を記録した
- [ ] owner / reviewerの責任境界を再確認した
- [ ] KPI impact（定量/定性）を記録した

## Governance Alignment
- [ ] Option A/Bと推奨案を明記した
- [ ] decision owner と decision due を設定した
- [ ] 次回review dateを設定した

## Outputs
- [ ] scope change report を生成した
- [ ] bundle check を実行した
- [ ] `docs/README.md` の関連リンクを更新した
- [ ] `docs/ticket-status.md` を更新した
