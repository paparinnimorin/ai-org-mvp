# PR Review Checklist (AI Org MVP)

## Consistency
- [ ] 承認ルール（1万円以上/外部公開/契約変更）が全ドキュメントで一致
- [ ] KPI名称と定義がテンプレ/仕様/チケットで一致
- [ ] RACIのAが重複していない

## Operability
- [ ] Daily/Weekly/Monthlyテンプレがそのまま使える
- [ ] 重大障害時の連絡テンプレが埋められる
- [ ] Runbookの責務分界（Ops vs Tech）が明確

## Safety
- [ ] Stop Conditions のトリガーが具体的
- [ ] 復旧Go/No-Go条件が明示されている
- [ ] 監査ログの記録先が明示されている
