# Governance Action Escalation Checklist

- [ ] Source minutes / action ID / reopen report を特定した
- [ ] Escalation level（L1/L2/L3）を設定した
- [ ] decision due（48時間以内）を設定した
- [ ] blocking factor と required decision を明記した
- [ ] `docs/reports/minutes/escalations/` にescalation reportを作成した
- [ ] action registerに `Escalated` を反映した
- [ ] close/reopenへの次フローを指定した
