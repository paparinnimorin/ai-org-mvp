# Approval Log Operations

## Fixed Format
- Date
- Request ID
- Requester
- Action Type
- Amount (JPY)
- Decision
- Approver
- Notes

## Owner & Timing
- 記録責任者: Division Lead（起案側）
- 監査責任者: PRIME AI
- 記録タイミング: 承認/差戻し直後（当日中）

## Monthly Audit Checklist
- 欠番IDがない
- 金額記録が妥当
- 承認者記録がある
- 重要案件の根拠が記録されている

## Delta Review
- 月次で前月との差分をレビュー
- 増加カテゴリを分析し、承認基準の見直し提案を作成
